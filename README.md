# Nucleo F103RB Base Source

## **Active Peripheral** <br><br>

### __GPIO__
>LED : PA5   

### **_USART2_**
>BPS : 2Mbps   
Rx : PA3   
Tx : PA2   

### **_USART3_**
>BPS : 2Mbps   
Rx : PC11 (Remap)   
Tx : PC10 (Remap)   

### **_I2C2_**
>Speed : FM (400Khz)   
SDA : PB11   
SCL : PB10   

### **_TIM1_**
>Mode : PWM1   
Period : 50Hz   
CH1 : PA8   
CH2 : -   
CH3 : -   
CH4 : -   

### **_TIM2_**
>Mode : PWM1   
Period : 20KHz   
CH1 : -   
CH2 : PA1   
CH3 : -   
CH4 : -
